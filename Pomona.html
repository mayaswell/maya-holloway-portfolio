Problem Statement

Small hobby farms or rural homesteads often rely on the sustained attention and effort of one to two individuals working in excess to survive. Increasingly erratic climate swings, large distances, and a lack of sustainable infrastructure inhibit commercial expansion into remote areas and some remote homesteads operate without standard utility connections or internet access.

Innovation Claim

Utilizing a local, low power edge processor, embedded sensors relay small packets of data at timed intervals through a long range (LoRa) radio mesh closed data pipeline to a home or gateway node. Focusing on energy efficiency and sustainable power solutions, these dispersed nodes provide a partially autonomous local agent with the information required to generate dashboards and reports for day to day data tracking, as well as notifications for specific, predetermined anomalies.

Project Description

This project focuses on creating a modular, low cost and low power solution to data collection and visualization in remote, dispersed farms and homesteads. After collecting data via embedded sensors, a low power local processor will convert the sensor data into smaller data ‘packets’ to be transmitted utilizing long range (LoRa) radio transmission to a ‘gateway’ or home node. Once the home node has stored the data to a local server, an open source LLM analyzes the data to generate timed reports and dashboards as well as notifications for specific, predetermined warnings.

Development Timeline

1-3 months: Finalize system architecture, parts list, power budget

3-4 months: Initial software design build and testing, begin integrating first sensor and formalizing data packets

4-5 months: First refactor; maximize data and power efficiency

5-7 months: Add additional local nodes for additional sensors/check for software modularity; initial testing of local agent

7-10 months: Advanced testing of agent/integration of predetermined decisions and choices to be made

10-12 months: Final stress testing and edge cases analysis, code and server tidying
